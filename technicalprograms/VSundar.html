Java Programming

Mrs. Vijayshree Sundar

Teacher Response:

What students will learn:

Confidence in Programming languages like HTML, CSS, Python, JavaScript, and Java

In-class Workload:

All work is done in class, learning, collaborating, and presenting

Homework:

No homework unless you are absent or missed class

Materials Required:

Notebook/journal, flash drive, headphones

Tests & Quizzes:

Quizzes at the end of each unit, pop quizzes occasionally, midterms in December and finals in June

Self-Studying:

Independent learning is highly encouraged, programming is more reliant on creativity and collaboration

If your class is AP, please describe the AP exam.

AP CS Principles is taken by Juniors, an easier AP. AP CS A is a rigorous AP, involves expertise in Java and debugging code

If your class is AP, please describe how the Honors and AP experience differs from a student's perspective.

There is no Honors designation for the Java technical program. Students are not required to take the AP tests at the end of the year but it is highly encouraged.
